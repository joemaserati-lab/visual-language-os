System / Typography

Typography is the first interface.

Redaction gives identity. Geist gives clarity. The system separates expressive display states from readable information states.

Rule Set

Every text style has a job.

Use semantic HTML for structure, then apply visual scale through tokens. The page should still make sense if all colors are removed.

H1One page statement.

Use one real h1 per page. It names the page argument, not just the page category.

H2Section decision.

Use h2 to mark a new decision layer: system rule, example group, specimen, method or validation block.

H3Module title.

Use h3 inside cards, examples and small repeated modules. Do not use it as a smaller H2.

PExplanation layer.

Paragraphs use Geist because they must be read quickly. Redaction is reserved for high-impact display and transformation states.

Body
Correct

Body copy explains the decision with enough vertical rhythm to let the reader move calmly from one line to the next.

Readable leading protects comprehension.
Wrong

Body copy explains the decision with too little vertical rhythm, forcing lines together and slowing down reading.

Compressed body text creates friction.
Body text uses readable line-height for sustained reading.
Small text
Correct

Metadata, captions and interface notes remain stable when their line-height is controlled.

Stable leading prevents small text from vibrating.
Wrong

Metadata, captions and interface notes drift apart when the line-height is too open.

Loose small text feels disconnected.
Small text needs stable line-height, especially in dense UI contexts.
Measure rules

Line length decides how fast text can be read.

Body copy
Correct

Body copy should stop around sixty-eight characters so the eye can return to the next line without losing position.

Maximum measure: circa 68ch.
Wrong

Body copy that stretches across the whole canvas makes every return sweep longer than necessary and turns an explanation into a horizontal endurance test.

Too wide for comfortable reading.
Use a maximum width near 68ch for normal paragraphs.
Lead
Correct

A lead introduces the thesis with a shorter, more deliberate measure.

Maximum measure: circa 52ch.
Wrong

A lead that runs too wide stops feeling like an opening thesis and starts behaving like an oversized paragraph.

Too broad for a lead statement.
Lead text should stay shorter than body copy, around 52ch.
Display titles
Correct

Sharp signal

Display titles work best around 8–12ch.
Wrong

This display title is much too long

Long display measures become decorative noise.
Keep display title measure short: about 8–12ch.

Forbidden typography

These choices break the reading contract.

Redaction 100 body
Correct

Use Geist for body text that must be read and understood.

Readable text stays in the text face.
Wrong

Never set paragraphs in Redaction 100.

Redaction 100 is not a body face.
Never use Redaction 100 for body copy.
Long abstract headings
Correct

Clear editorial heading

Readable display fonts can carry longer headings.
Wrong

Long headings should not become abstract puzzles

Abstract display states must stay brief.
Do not set long headings in overly abstract display fonts.
Uppercase paragraphs
Correct

Use sentence case for paragraphs so word shapes stay recognizable.

Paragraphs need natural rhythm.
Wrong

Do not turn a full paragraph into uppercase text because it removes natural word shape and slows reading.

Uppercase is for labels, not paragraphs.
Never use long uppercase paragraphs.
Cramped body
Correct

Body text needs enough line-height to support scanning, comprehension and return sweeps.

Readable body leading protects the reader.
Wrong

Body text with overly compressed line-height becomes dense, noisy and difficult to read over multiple lines.

Do not compress body text.
Never use overly compressed line-height on body text.

Do / Don't

Do

Use Redaction 35 for readable display.

Large editorial headlines, short statements and brand moments stay readable while still feeling distinct.

Don't

Use Redaction 100 for body copy.

Abstract states belong in motion, texture or transition moments. They are not final reading states.
