System / Typography

Typography is the first interface.

Redaction is the display and editorial layer. Geist is the reading and interface layer. Body copy, paragraphs, navigation, labels and instructions always use Geist.

Rule Set

Every text style has a strict job.

Use semantic HTML for structure, then apply font roles with discipline. Redaction creates identity. Geist carries information.

H1One page statement.

Use one real h1 per page. It names the page argument, not just the page category.

H2Section decision.

Use h2 to mark a new decision layer: system rule, example group, specimen, method or validation block.

H3Module title.

Use h3 inside cards, examples and small repeated modules. Do not use it as a smaller H2.

PReading layer.

Paragraphs, body copy, labels, navigation and instructions use Geist. Redaction must not be used for continuous reading.

Italic Voice

Italic creates editorial tension.

Use Redaction Italic as an accent voice, not as automatic emphasis. It should interrupt the rhythm with intention.

A shift in posture can become a second voice.

I-001Italic is an accent, not a default style.

Use it for short editorial insertions, quoted fragments, tension words and expressive page moments.

I-002Functional text stays Geist.

Navigation, buttons, labels, instructions and body copy must remain immediately readable in Geist.

Reading Layer

Geist carries the body.

Any text that asks the user to read, decide, navigate or act belongs to Geist. Redaction should never be the default bodycopy voice.

Do / Don't

Do

Use Redaction 20 for XXL display.

Hero-scale titles need a cleaner Redaction state so the identity stays expressive without becoming overly pixelated.

Don't

Use Redaction 100 for body copy.

Abstract states belong in motion, texture or transition moments. They are not final reading states.

Do

Use Geist for every reading layer.

Paragraphs, bodycopy, lead copy, navigation, labels, captions, buttons and instructions stay in Geist.

Don't

Use Redaction for paragraphs.

Redaction is too expressive for continuous reading and should not carry bodycopy or functional interface text.

Do

Use Redaction Italic as a second voice.

Short italic accents can create contrast, attitude and editorial rhythm inside an otherwise disciplined page.

Don't

Use italic for interface instructions.

Functional text must stay stable, upright and instantly readable in Geist.
