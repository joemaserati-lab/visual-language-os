System / Typography

Typography is the first interface.

Redaction gives identity. Geist gives clarity. Their roles must never be confused.

Typeface roles

T-001 / GeistInformation layer.

Use Geist for body text, navigation, UI, captions, metadata, forms and anything that must be read quickly.

T-002 / Redaction 35Readable display layer.

Use Redaction 35 for hero titles, section statements and editorial moments where identity matters but readability must remain intact.

T-003 / Redaction 50Threshold layer.

Use Redaction 50 only for short words, accents or transition states. It is not safe for long content.

T-004 / Redaction 70 + 100Abstract layer.

Use Redaction 70 and 100 only as abstract or transition states. They are not final readable typography.

Visual scale

Display XL → Display → H1 → H2 → H3 → Lead → Body → Small

Accessibility and SEO use correct headings. Visual rhythm uses display tokens.
