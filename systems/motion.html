System / Motion

Motion must reveal information.

Motion is not decoration. It is a sequencing layer that clarifies hierarchy, state, rhythm and attention.

Motion laws

M-001Motion must have a job.

Reveal, connect, clarify, orient or confirm. If it does none of these, remove it.

M-002Readability must arrive early.

Scroll reveals must complete before the element reaches the central focal zone.

M-003No opacity masking for font-state transitions.

If the purpose is to show transformation, opacity should not hide the effect.

Timing bands

120ms micro / 240ms hover / 480ms reveal / 900–1400ms scroll transformation
