System / Motion

Motion must reveal information.

Motion is not decoration. It is a sequencing layer that clarifies hierarchy, state, rhythm and attention.

Motion Laws

Movement needs a reason.

M-001Motion must have a job.

Reveal, connect, clarify, orient or confirm. If it does none of these, remove it.

M-002Readability must arrive early.

Scroll reveals complete before the element reaches the central focal zone.

M-003Transform state, not attention.

The user should understand what changed, not just notice that something moved.

Do / Don't

Do

Use motion to expose sequence.

A section can move from abstract to readable when the transformation reinforces discovery.

Don't

Use motion to hide weak hierarchy.

If the page needs animation to become interesting, the static composition is not strong enough.
