System / Composition

Space gives the interface a voice.

Composition controls pace, focus and confidence. Use spacing, grid, measure and alignment as active design decisions, not as leftover layout details.

01 / Section spacing

Sections need enough distance to reset attention.

Vertical padding marks a meaningful change in topic. Tight sections feel accidental because the reader cannot tell whether content belongs to the previous idea or begins a new one.

Too little padding Section title

The content starts before the eye has had time to reset.

Correct spacing Section title

Generous vertical padding creates a clean opening and a deliberate pause.

Do

Use large section padding for new ideas.

Give each major block a clear entrance and exit.

Don't

Stack unrelated sections too tightly.

Crowded transitions make hierarchy feel weak.

02 / Vertical rhythm

Repeated spacing creates reading momentum.

Use a predictable rhythm between labels, headings, paragraphs and actions. Rhythm should be varied only when the meaning changes.

Do

Build from the spacing scale.

Use repeated intervals so modules feel related.

Don't

Mix arbitrary gaps.

Random spacing makes the page feel uncomposed.

03 / Grid usage

The 12 column grid is a relationship system.

Use columns to define proportion, not decoration. Combining spans makes asymmetry controlled: span-4 for compact modules, span-5 and span-7 for editorial balance.

span-5
span-7
Do

Use spans to express priority.

Let important content occupy wider relationships.

Don't

Fill columns just because they exist.

The grid should clarify, not create noise.

04 / Content width

Measure protects comprehension.

Text blocks need a readable line length. Wide paragraphs slow scanning because the eye travels too far before returning to the next line.

Correct measure

Readable content width keeps the paragraph calm, focused and easy to scan across multiple lines without losing the next starting point.

Too wide

When a paragraph stretches across the full available canvas, the line length becomes tiring and the reader must work harder to track each return from the end of one line to the beginning of the next.

Do

Keep body copy near a readable measure.

Use width constraints for paragraphs and notes.

Don't

Let long text fill every container.

Full-width prose reduces reading comfort.

05 / Negative space

Empty space is how importance becomes visible.

Negative space separates the primary message from supporting details. It gives the eye a place to rest before it commits to the next action.

Focus
Do

Leave intentional empty areas.

Space can be the strongest hierarchy cue.

Don't

Fill every visible gap.

Overfilling removes focus and weakens contrast.

06 / Density control

Density should match the decision state.

Dense cards are useful for comparison and scanning, but they should not make the reader fight for meaning. Add internal spacing when content needs reflection.

Too denseSystem note

Labels, title, body and actions are squeezed into one block.

Correct spacingSystem note

Each layer has room, making the same information easier to understand.

Do

Open spacing around complex content.

Give each element a recognizable role.

Don't

Compress every card to save space.

Efficiency without readability creates friction.

07 / Alignment

Alignment makes separate elements feel designed together.

Shared edges create trust. Break alignment only when the break has a clear visual or narrative purpose.

Do

Share edges across related content.

Aligned starts and ends reduce visual effort.

Don't

Let modules drift without reason.

Misalignment reads as accidental before it reads as expressive.

08 / Visual breathing room

Interfaces need pauses between moments of emphasis.

Breathing room is the combination of padding, line-height, margins and quiet surfaces. It prevents every element from competing at the same volume.

Calm surface

One clear idea at a time.

When the surrounding space is quiet, the message can be strong without becoming loud.

Do

Use calm areas around focal content.

Let important moments breathe before the next module begins.

Don't

Put every element in competition.

Constant emphasis makes emphasis disappear.
